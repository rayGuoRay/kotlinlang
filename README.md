### 编译运行
具体使用方式请参考<<[使用命令行编译运行 Kotlin](http://blog.csdn.net/liangjiu2009/article/details/72518558)>>一文,简单编译运行命令如下:

- 编译 : kotlinc hello.kt -include-runtime -d hello.jar
- 运行 : java -jar hello.jar
